## Detailed Findings

# Detailed Findings

### Cashback Redemption Confusion — 43.75% of Rewards Calls
* Primary driver: Digital (deeply nested preference setting, no 'Set as default' option).
* Contributing: Communication (lack of clear guidance/tutorials).
* Preventability: 100% — streamline preference setting, add auto-redemption, provide in-app tutorials.
* Recommended: Streamline redemption preference setting to a top-level rewards setting, add a 'Set as default' option, and allow auto-redemption rules. Provide in-app tutorials and proactive messages to guide users on managing preferences.
![Chart](data/multi_lens_breakdown.png)

### Welcome Bonus Tracking Anxiety — 12.5% of Rewards Calls
* Primary driver: Digital (no real-time tracker, updates only per statement cycle).
* Contributing: Communication (bonus terms in fine print, hard to read on mobile).
* Preventability: 100% — implement real-time tracker, clear terms, push notifications.
* Recommended: Implement a real-time welcome bonus spend tracker with a progress bar and days remaining. Clearly list qualifying/non-qualifying purchases. Send push notifications for progress updates and present bonus terms in a clear, mobile-friendly format.
![Chart](data/multi_lens_breakdown.png)

### Merchant Category Code Confusion — 18.75% of Rewards Calls
* Primary driver: Digital (invisible MCCs, no mechanism to report incorrect coding).
* Contributing: Communication (unclearly explained category bonus exclusions).
* Preventability: 100% — display MCCs, allow flagging, clear exclusions.
* Recommended: Display MCC and assigned category for each transaction, allow customers to flag incorrect categorization, and clearly list category bonus exclusions with examples. Add in-app explainers for MCCs and their impact.
![Chart](data/multi_lens_breakdown.png)

### Unexpected Tier Downgrades — 6.25% of Rewards Calls
* Primary driver: Communication (no proactive notification before downgrade).
* Contributing: Digital (mobile-unoptimized page, no progress bar for next tier).
* Preventability: 100% — update mobile page, implement advance notifications.
* Recommended: Update the tier status page for mobile with a clear progress bar. Implement 90-day and 30-day advance notification system (email/push) for tier downgrades, clearly stating current/new status and affected benefits.
![Chart](data/multi_lens_breakdown.png)

### Lack of Real-time Bonus Tracking — 6.25% of Rewards Calls
* Primary driver: Digital (no real-time visibility into category bonus tracking/earn rates).
* Contributing: Communication (no in-app tips on maximizing rewards).
* Preventability: 100% — show merchant earning rate preview, bonus category progress.
* Recommended: Show merchant earning rate preview on each transaction, display bonus category progress on the main rewards dashboard, and add a real-time earn rate indicator on transaction detail screens with a qualifying merchant badge. Provide in-app tips on maximizing rewards.
![Chart](data/multi_lens_breakdown.png)
